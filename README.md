# Лендинг — Антон Леонов, частный риелтор (агентство ВамКнам)

Одностраничный сайт. Тёмный кинематографичный hero + светлые секции, золотой акцент.
Чистый HTML/CSS/JS, без сборки. Реализация дизайна из Claude Design.

## Как открыть локально
- **Просто:** дважды кликните `index.html`.
- **Через сервер (рекомендуется):** в терминале в этой папке `python3 serve.py`,
  затем откройте `http://localhost:5180`.

Шрифты (Unbounded, Manrope) подключаются с Google Fonts — нужен интернет.
Остальное (фото, картинки) лежит локально в `assets/`.

## Файлы
- `index.html` — разметка и текст.
- `styles.css` — оформление (палитра и шрифты в начале файла).
- `script.js` — кинематографичный hero, меню, FAB, аккордеон, появления при скролле.
- `assets/`:
  - `anton.jpg` — фото риелтора, `vamknam-mark.png` — лого агентства (и favicon).
  - `hero-room.jpg` — фон hero.
  - `obj-*.jpg` — фото категорий в блоке «Объекты» (иллюстрации, можно заменить своими).

## Публикация на GitHub Pages
1. Создайте на github.com новый **публичный** репозиторий (например `rieltor-landing`),
   без README.
2. В терминале в этой папке выполните (репозиторий уже инициализирован и закоммичен):
   ```
   git remote add origin https://github.com/ВАШ_ЛОГИН/rieltor-landing.git
   git branch -M main
   git push -u origin main
   ```
3. На GitHub: **Settings → Pages → Build and deployment → Source: Deploy from a branch**,
   ветка `main`, папка `/ (root)`. Сохраните.
4. Через 1–2 минуты сайт будет доступен по адресу
   `https://ВАШ_ЛОГИН.github.io/rieltor-landing/`.
